# fractol
Fractal Explorer
![Exemple](https://raw.githubusercontent.com/Denis2222/fractol/master/gfx/screen.png)

---

Compile:
```
make
```
Run:
```
./fractol <fractal_number>
```
1 Mandelbrot

2 Julia

3 Mandelbrot (Z^3 + C)

4 Mandelbrot (Z^2 + C^3)

5 Julia (Z^3 + C^3)

6 Julia Custom (cos)

7 Julia (Z^5 + C)

8 Pentagon de durer

---

Move : Arrow

Zoom: {W-S}

Iter: {A-D}

Change fractal: {[-]}

Mouse Wheel: Zoom

Mouse Click: Center to click

Enable/Disable move: {F} + mouse move

Export FdF: {L}
